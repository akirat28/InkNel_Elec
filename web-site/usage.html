Features

機能一覧

InkNel に搭載されている主な機能をカテゴリ別にまとめています。 すべての機能はローカル動作で、データは .md ファイルとして保存されます。

01

エディタ & プレビュー

Markdown エディタ

CodeMirror 6 ベース。見出し / リスト / 引用 / コードブロックを構造的に編集できます。

シンタックスハイライト

JavaScript / Python / Go / Rust など 40 以上の言語に対応。コードブロックはコピー可能。

検索 / 置換

ノート内検索・置換に加えて、全ノート横断のタイトル / 全文検索もサポート。

タブ & 分割

複数ノートをタブで切替。エディタ / プレビューはタブごとに独立して切替可能。

02

整理 & ナビゲーション

フォルダ階層

仮想フォルダでノートを階層管理。ドラッグでフォルダ移動も可能。

タグ

ノートに任意のタグを付けて横断的に分類。タグ単位でフィルタ表示できます。

ノート間リンク

ノート同士を双方向にリンク。関連メモを辿って思考を広げられます。

テンプレート

定型ノートを雛形として登録。本文だけでなくタグまで継承して新規ノートを作成。

03

AI アシスト

マルチプロバイダ連携

ChatGPT (OpenAI) / Claude (Anthropic) / Gemini (Google) など複数の AI を切替可能。

ストリーミングチャット

応答はストリーミング表示。長い回答も最初の数語からすぐに読み始められます。

ノート操作モード

チャットから「新規ノート作成 / 既存ノートに追記 / 全体書き換え」を AI に指示。

AI 指示パターン

頻用するプロンプトを定型として登録。1 クリックで AI に流し込めます。

04

プラグイン拡張

プラグインストア

リモートカタログから機能を後付け。設定画面からインストール / 無効化を切替。

カレンダープラグイン

日付ベースのノート作成。祝日 / 予定の表示、既存ノートのバッジ表示もサポート。

Mermaid 図

フローチャート / シーケンス図 / ガントチャートをコードブロックで描画。

05

保存 & セキュリティ

ローカル保存

すべてのノートは .md ファイルとしてローカルに保存。YAML front-matter にメタ情報を埋め込み、別環境への持ち運びも自在。

ZIP バックアップ

保存先フォルダごとワンクリックで ZIP 化。リストアで完全復元できます。

ノート保護

個別ノートにパスワードを設定。閲覧時にロック解除を求めるシークレットモード対応。

クラウド同期 (Beta)

Google Drive 等のクラウドストレージとの同期をサポート（プロバイダ切替可）。

06

外観 & 多言語

ダーク / ライト

2 種のテーマを瞬時に切替。長時間執筆でも目に優しい配色。

フォント設定

本文用 / サイドバー用のフォントとサイズを個別に指定可能。

日本語 / 英語 UI

UI 表示言語を日本語 / 英語で切替。AI チャットも同言語で応答するよう連動。

How to use

使い方

インストール後、最初の数分で押さえておきたい基本操作を順に解説します。

01

インストール & 起動

ダウンロードページ から最新の .dmg を取得し、Applications フォルダ にドラッグ＆ドロップしてください。 初回起動時は macOS の「App を開いてもよろしいですか？」確認に許可を与えると以降はそのまま起動できます。

Note: Apple Notarization 済みの正規ビルドのため、Gatekeeper の警告は表示されません。
02

新しいノートを作る

サイドバー上部の 「+」 アイコン、またはメニューの ファイル → 新規ノート （⌘ N）でノートを追加できます。タイトル行にカーソルが入った状態で開くので、そのまま入力 → Tab で本文へ。

  • サイドバーでフォルダを選択した状態で新規作成 → そのフォルダの中に作成
  • ノート右上の タグバー でタグを付与
  • 本文は .md ファイルとして即時保存（手動セーブ不要）
03

フォルダ & タグで整理する

サイドバーで右クリック → 新規フォルダ から階層を作成。ノートはドラッグで移動できます。 タグはノート上部のタグバーからカンマ区切りで追加。複数タグでの横断検索も可能です。

  • フォルダのリネーム / 削除 はサイドバー右クリックメニューから
  • 削除時、配下のノートはまとめてゴミ箱へ（.md ファイルも一緒に削除）
04

エディタ / プレビューを切り替える

タブのモード切替ボタンで エディタ ⇄ プレビュー を切替。 コードブロックは言語名を指定すると 40 以上の言語でハイライトされ、右上のボタンでクリップボードコピーできます。

Tip: プレビュー内のチェックボックス - [ ] はクリックでトグル可能。 編集中のノートに即座に反映されます。
05

AI チャットを使う

アクティビティバーの AI アイコン でチャットパネルを開きます。 初回は 設定 → AI で利用するプロバイダ (ChatGPT / Claude / Gemini) と API キーを登録してください。

  • 「チャットモード」 — 通常の対話。回答はストリーミング表示
  • 「編集モード」 — 開いているノートに対して追記 / 書き換え / 新規作成を AI に依頼
  • AI 指示パターン を定型として登録すれば、1 クリックで定型プロンプトを送信
06

テンプレートを活用する

設定 → テンプレート で指定したフォルダ配下のノートが、エディタツールバーの テンプレートボタン から呼び出せるようになります。 本文だけでなくタグ情報もそのまま継承するので、日報・議事録・読書ログなどの定型運用に便利です。

07

プラグインを追加する

設定 → プラグイン からリモートカタログを開き、必要な機能を取り込みます。 プラグインは独立した JavaScript モジュールとして動作するため、有効化 / 無効化はワンクリック。

  • カレンダー — 日付ベースのノート作成 + 祝日表示
  • Mermaid — コードブロックからフローチャート等を描画
  • 開発者モード — ローカルフォルダの未公開プラグインを直接読み込み可能
08

バックアップ & リストア

設定 → バックアップ から保存先フォルダ全体を ZIP に圧縮できます。 別の Mac へ移行する場合や、定期スナップショットを残しておきたい場合に活用してください。 リストア 機能で ZIP を選ぶと、その時点の状態に完全復元します。

Note: リストアは既存データを上書きします。実行前に現状の ZIP を取得しておくと安全です。
09

ノートを保護する

機密ノートは 保護フラグ を ON にして個別パスワードを設定。 アプリ起動後、初めて開くときに PIN 入力を求められ、セッション中は同じノートを何度開いてもロック解除済みのままです。

  • シークレットノート — 一覧から非表示にする強い保護モード
  • 保護ノートは削除不可。保護を外してから削除する設計
