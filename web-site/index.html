AI × Markdown

構造化されたテキストは
AI と相性が抜群

Markdown は見出し・リスト・引用・コードブロックといった「意味の構造」をそのまま記述できる軽量フォーマット。AI モデルは構造化された文章を読みやすく、編集しやすいので、執筆を支援するパートナーとして自然に組み込めます。

InkNel は ChatGPT / Claude / Copilot と連携し、要約・整形・対話・続きの提案までエディタの中で完結します。書きながら磨く、磨きながら書く。

要点まとめ

長い議事録や記事から要点だけを抽出してリスト化。

構成の提案

ドラフトを渡せば見出しレベルや章立てを再構築。

文章作成サポート

下書きから本文展開、語調や表現の言い換えまで。

タスクリスト整理

箇条書きの混じった文章から TODO を自動抽出。

表の整形

CSV や箇条書きから Markdown table へ自動変換。

HTML → Markdown

貼り付けた HTML を綺麗な Markdown に変換。

How it works

3 ステップで
使い始められます

01

ダウンロード & インストール

macOS 用の DMG をダウンロード。アプリを Applications フォルダにドラッグするだけ。

02

保存先を設定

任意のフォルダを保存先に指定。ノートはすべて .md として書き出されます。

03

書き始める

新規ノートを作って、Markdown で執筆。AI 要約やプラグインも好みに合わせて。

メモを書く
