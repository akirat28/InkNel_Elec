# InkNel 画面構成リファレンス

このドキュメントは InkNel の UI 構成を整理したもので、**「○○エリアの□□に××を追加して」のような指示を出すときの参照** として使用する。各エリアには **正式名称**（このドキュメントで使う呼び方）を定めている。

---

## 1. 全体レイアウト

ウィンドウ全体は **横方向の3カラム** + メインに **縦方向の積層**。

```
┌──────┬──────────────┬─────────────────────────────────┐
│      │              │                                 │
│  A   │              │   メイン領域 (Main Area)        │
│ ク   │              │                                 │
│ テ   │   サイドバー │  ┌────────────────────────────┐ │
│ ィ   │ (Sidebar)    │  │ NoteHeader                 │ │
│ ビ   │              │  ├────────────────────────────┤ │
│ テ   │              │  │ EditorToolbar              │ │
│ ィ   │              │  ├────────────────────────────┤ │
│ バ   │              │  │                            │ │
│ │    │              │  │   Editor / Preview         │ │
│      │              │  │                            │ │
│      │              │  │                            │ │
│      │              │  └────────────────────────────┘ │
└──────┴──────────────┴─────────────────────────────────┘
  48px    可変幅 (160-480px)            残り全幅
```

| ID | 名称 | 幅 | 役割 |
|---|---|---|---|
| A | **アクティビティバー** (`ActivityBar`) | 48px 固定 | ビューモード切替（ファイル一覧 / 検索 / タグ / 設定） |
| B | **サイドバー** (`Sidebar`) | 可変 (160〜480px) | ファイル一覧ツリー or 検索パネル or タグパネル |
| C | **メイン領域** (`Main Area`) | 残り全幅 | 選択中ノートの表示・編集 |

サイドバーの右端ドラッグハンドルで幅を調整可能。アクティビティバーの「ファイル」「検索」「タグ」アイコンを再クリックでサイドバーを折りたたみ。

## 2. アクティビティバー (`ActivityBar`)

画面の最左端、48px 幅の縦アイコン列。VS Code 風。

```
┌──────┐
│      │
│  📄  │ ← ファイル
│      │
│  🔍  │ ← 検索
│      │
│  🏷  │ ← タグ
│      │
│  ⚙   │ ← 設定（最下部）
│      │
└──────┘
```

| アイコン | 名称 | クリック動作 |
|---|---|---|
| 📄 | **ファイルアイコン** | サイドバーをファイルモードに切替 / 既にファイルモードならサイドバー折りたたみトグル |
| 🔍 | **検索アイコン** | サイドバーを検索モードに切替 / 既に検索モードなら折りたたみトグル |
| 🏷 | **タグアイコン** | サイドバーをタグモードに切替 / 既にタグモードなら折りたたみトグル |
| ⚙ | **設定アイコン**（最下部固定） | 設定モーダルを開く（サイドバーモードは変更しない） |

**ファイル**: `src/components/ActivityBar.tsx`
**CSS クラス**: `.activity / .activity__btn / .activity__btn.is-active`

## 3. サイドバー (`Sidebar`)

中央のカラム。アクティビティバーの選択により2モードを切り替え。

### 3.1 共通要素

```
┌──────────────┐
│ ファイル  📄+📁+│ ← サイドバーヘッダー (sidebar__header)
├──────────────┤
│              │
│   本体       │
│              │
│              │
└──────────────┘
              ↑右端: ドラッグハンドル (sidebar__resizer)
```

| 要素 | 名称 | 備考 |
|---|---|---|
| ヘッダーのタイトル | **サイドバータイトル** | "ファイル" or "検索" |
| 右端のリサイズハンドル | **サイドバーリサイザー** | 左右ドラッグで幅変更 |

### 3.2 ファイルモード (mode = `'files'`)

```
┌──────────────┐
│ ファイル 📄+📁+│ ← 「+ ボタン群」: 📄+ 新規ノート / 📁+ 新規フォルダ
├──────────────┤
│ ▼ 📁 階層1     │ ← フォルダ行 (tree__folder)
│   📄 ノートA  ⋮│ ← ファイル行 (tree__file)、⋮はホバー時のみ
│   ▼ 📁 階層2  ⋮│
│     📄 ノートB🔒⋮│ ← 🔒は保護中 (tree__lock-indicator)
│ 📄 はじめに   ⋮│
└──────────────┘
```

| 要素 | 名称 | 場所 |
|---|---|---|
| 📄+ ボタン | **新規ノートボタン** | sidebar__header 右側 |
| 📁+ ボタン | **新規フォルダボタン** | sidebar__header 右側 |
| ▼/▶ | **チェブロン** (tree__chevron) | フォルダ行先頭、展開状態 |
| 📁 アイコン | **フォルダアイコン** (tree__icon) | フォルダ行 |
| 📄 アイコン | **ファイルアイコン** (tree__icon) | ファイル行 |
| ⋮ ボタン | **ケバブボタン** (tree__menu-btn) | 行右端、ホバー時のみ表示 |
| 🔒 アイコン | **錠前インジケータ** (tree__lock-indicator) | 保護中ファイルの右端 |

**ケバブメニュー（ファイル）の項目**:
1. ✏️ 名称変更
2. 🔒/🔓 保護 / 保護解除
3. 🗑 削除（保護中はグレーアウト）

**ケバブメニュー（フォルダ）の項目**:
1. ✏️ 名称変更（配下を一括更新）

**操作**:
- ファイル → フォルダへ **ドラッグ&ドロップで移動**
- フォルダの ▼ クリックで展開/折りたたみ

### 3.3 検索モード (mode = `'search'`)

```
┌──────────────┐
│ 検索          │
├──────────────┤
│ [入力欄][検索]│ ← 検索フォーム (search-panel__form)
├──────────────┤
│ 3 件         │
│ ┌──────────┐ │
│ │ノートA  🔒│ │ ← 検索結果アイテム
│ │work/proj │ │
│ └──────────┘ │
│ ┌──────────┐ │
│ │別ノート   │ │
│ └──────────┘ │
└──────────────┘
```

| 要素 | 名称 | 備考 |
|---|---|---|
| 入力欄 | **検索入力欄** | ↑↓ で過去の履歴をたどれる |
| 検索ボタン | **検索実行ボタン** | Enter でも実行可 |
| 結果リスト | **検索結果リスト** | タイトル一致 → 本文一致の順 |

### 3.4 タグモード (mode = `'tags'`)

```
┌──────────────┐
│ タグ          │
├──────────────┤
│ ▶ 🏷 #important  3 │ ← タグ行 (tags-list__row) 折りたたみ状態
│ ▼ 🏷 #todo       2 │ ← 展開状態
│     📄 ノートA       │ ← 配下のノート行 (tree__file)
│     📄 ノートB       │
│ ▶ 🏷 #idea       5 │
└──────────────┘
```

| 要素 | 名称 | 備考 |
|---|---|---|
| タグ行 | **タグ行** (tags-list__row) | クリックでアコーディオン展開/折りたたみ |
| 件数バッジ | **タグ件数バッジ** (tags-list__count) | そのタグを含むノート数 |
| 配下のノート行 | **タグ配下ノート行** (tree__file) | クリックでノートを開く |

**タグの定義**:
- 本文中の `#word` 記法（`word` は文字/数字/`_`/`-`、Unicode 対応）
- 行頭または空白の直後にある `#` のみ対象
- マークダウンの**見出し** (`# `〜`###### `) と fenced **コードブロック**内は対象外
- 全ノートの本文を main プロセスでスキャンして集計（`notes:list-tags` IPC）

**ファイル**:
- `src/components/Sidebar.tsx` (本体・ツリー描画)
- `src/components/SearchPanel.tsx` (検索モード)
- `src/components/TagsPanel.tsx` (タグモード)
- `src/components/ContextMenu.tsx` (kebab メニュー)

## 4. メイン領域 (`Main Area`)

選択中のノートを表示・編集する右側の領域。**縦に3つのブロック**を積層。

```
┌────────────────────────────────────────┐
│ NoteHeader                             │ ← 4.1
├────────────────────────────────────────┤
│ EditorToolbar (編集モード時のみ)      │ ← 4.2
├────────────────────────────────────────┤
│ TagBar (編集モード時のみ)             │ ← 4.3
├────────────────────────────────────────┤
│                                        │
│   Editor (編集)  または  Preview      │ ← 4.4
│                                        │
└────────────────────────────────────────┘
```

### 4.1 ノートヘッダー (`NoteHeader`)

```
┌────────────────────────────────────────┐
│ [ファイル名 (例: 階層1/テスト1)] [✎][👁]│
└────────────────────────────────────────┘
```

| 要素 | 名称 | 役割 |
|---|---|---|
| 入力欄 | **ファイル名入力欄** (note-header__name) | スラッシュ区切りパス入力。例: `階層1/テスト1` |
| ✎ ボタン | **編集トグル** (view-toggle__btn) | 編集ビューに切替 |
| 👁 ボタン | **プレビュートグル** (view-toggle__btn) | プレビュービューに切替 |

ファイル名はスラッシュで分割され、最後のセグメントが title、それ以前が folder として保存される。

**ファイル**: `src/components/NoteHeader.tsx`

### 4.2 編集ツールバー (`EditorToolbar`) — 通称「タグボタンリスト」

編集ビュー時のみ NoteHeader 直下に表示される、マークダウン挿入ボタンの一覧。
**4つのグループ** を区切り線で分けて配置。

```
┌──────────────────────────────────────────────────────────────────────┐
│ H1 H2 H3 H4 H5 H6 │ B I S <> │ • 1. ☐ ❝ │ 🔗 {} ― ▦ 😊             │
└──────────────────────────────────────────────────────────────────────┘
   見出しグループ    強調グループ  リストグループ  挿入グループ
```

| グループ名 | 含まれるボタン | 動作 |
|---|---|---|
| **見出しグループ** | H1〜H6 | 行頭に `#` 〜 `######` を追加 |
| **強調グループ** | **B** (太字) / *I* (斜体) / ~~S~~ (取消) / `<>` (コード) | 選択範囲を記号で囲む |
| **リストグループ** | • (箇条書き) / 1. (番号) / ☐ (タスク) / ❝ (引用) | 行頭に各 prefix を追加 |
| **挿入グループ** | 🔗 (リンク) / `{}` (コードブロック) / ― (区切り線) / ▦ (テーブル) / 😊 (アイコン) | カーソル位置に各記法を挿入 |

**挿入グループの特殊ボタン**（吹き出し型ポップアップを開く）:
- **▦ テーブルボタン** → `TablePicker`（8×8 グリッドで行×列を選択）
- **😊 アイコンボタン** → `IconPicker`（カテゴリ別の絵文字一覧）

**ファイル**:
- `src/components/EditorToolbar.tsx` (本体)
- `src/components/TablePicker.tsx`
- `src/components/IconPicker.tsx`

**CSS**: `.md-toolbar / .md-toolbar__group / .md-toolbar__divider / .md-toolbar__btn`

### 4.3 タグバー (`TagBar`)

EditorToolbar 直下、編集ビュー時のみ表示。ノートにメタデータとして紐づくタグをカンマ区切りで入力できる。

```
┌────────────────────────────────────────┐
│ 🏷 [#todo ×] [#work ×] [入力欄........] │
└────────────────────────────────────────┘
```

| 要素 | 名称 | 役割 |
|---|---|---|
| 🏷 アイコン | **タグバーアイコン** (tag-bar__icon) | 左端のラベル |
| バッジ | **タグバッジ** (tag-bar__badge) | 確定済みタグ。右の `×` で削除 |
| 入力欄 | **タグ入力欄** (tag-bar__input) | カンマで確定、Enter でも確定、空入力 BS で末尾削除 |

**確定ルール**:
- カンマ (`,` または `、`) を入力した瞬間、その手前の文字列が新しいタグになる
- Enter キーでも確定（IME 変換中の Enter は無視）
- 空入力でバックスペース → 末尾のタグを削除
- 重複タグは自動的に除外
- フォーカスアウト時にも未確定の draft をコミット

**保存**:
- ノートメタデータ (`notes.tags` カラム、JSON 配列) として SQLite に保存
- `notes:list-tags` IPC で本文中の `#word` タグと統合され、サイドバーのタグモードに表示される

**ファイル**: `src/components/TagBar.tsx`

### 4.4 編集 / プレビュー領域

| ビュー | コンポーネント | 内容 |
|---|---|---|
| 編集ビュー | `Editor` (CodeMirror 6) | マークダウンソース編集 |
| プレビュービュー | `Preview` (markdown-it) | レンダリング結果表示。先頭に **タグバッジ列**（`preview__tags`）があり、TagBar で設定したタグを `#tag` 形式で表示 |

**ドラッグ&ドロップ機能（編集ビューのみ）**:
- 画像 (PNG/JPG/...) → `images/<sha256>.<ext>` に保存 + 自動挿入
- PDF → `attachments/` 保存 + サムネイル付き挿入
- ZIP/LZH/7z → `attachments/` 保存 + 📎 リンクとして挿入

**プレビューでの操作**:
- 画像クリック → ライトボックスで拡大表示
- 添付リンククリック → OS の既定アプリで開く
- http(s) リンククリック → 既定ブラウザで開く

**ファイル**:
- `src/components/Editor.tsx`
- `src/components/Preview.tsx`

## 5. モーダル / ダイアログ / ポップアップ

メイン領域とは別レイヤーに表示される UI 要素。

| 名称 | 開く方法 | 内容 |
|---|---|---|
| **設定モーダル** (`PreferencesModal`) | アクティビティバーの ⚙ / メニュー / `⌘,` | 2ペイン構成: 「基本」「保護」 |
| **パスワードダイアログ** (`PasswordDialog`) | 保護中ノートで編集に切り替えた時 | 4桁の数字パスワード入力 |
| **名称変更ダイアログ** (`RenameDialog`) | ケバブメニューの「名称変更」 | ファイル/フォルダ名の編集 |
| **画像ライトボックス** | プレビューの画像クリック | フルスクリーン暗背景で拡大表示 |
| **コンテキストメニュー** (`ContextMenu`) | ファイル/フォルダ行の ⋮ クリック | 吹き出し型ポップアップ |
| **テーブルピッカー** (`TablePicker`) | 編集ツールバーの ▦ クリック | 8×8 のサイズ選択グリッド |
| **アイコンピッカー** (`IconPicker`) | 編集ツールバーの 😊 クリック | カテゴリ別絵文字一覧 |

### 5.1 設定モーダル (`PreferencesModal`)

```
┌─────────────────────────────────────┐
│ 設定                              × │
├──────────┬──────────────────────────┤
│ ▶ 基本   │  右ペインに詳細フォーム  │
│   保護   │                          │
└──────────┴──────────────────────────┘
```

| カテゴリ | 設定項目 |
|---|---|
| **基本** | テーマ (ダーク/ライト) / 編集ボタンの表示 / 検索履歴の保存方式 / 検索履歴の件数 |
| **保護** | パスワード（4桁数字、初期値 1234） |

**ファイル**: `src/components/PreferencesModal.tsx`

## 8. 指示を出すときの例

このリファレンスを使って、次のような指示が明確に伝わる:

- 「**編集ツールバーの挿入グループ** にカラーピッカーボタンを追加して」
- 「**サイドバーのファイルモード** で、フォルダの **ケバブメニュー** に "新しいサブフォルダ" を追加して」
- 「**ノートヘッダー** の右側、**編集/プレビュートグル** の左にお気に入りボタンを置いて」
- 「**設定モーダル** に新しいカテゴリ "外観" を追加して、その下にフォントサイズ設定を置いて」
- 「**サイドバーの検索モード** の **検索結果リスト** に、本文の抜粋（マッチ周辺）を表示して」
